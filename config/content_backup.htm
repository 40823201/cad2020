使用 cmsimde 建立靜態網誌:

1. 在 github 建立倉儲[CAD2020]，在git clone 到近端

2. 之後再加入除了 cmsimde 目錄外的所有內容以 git submodule add (如老師筆記所記)，建立 cmsimde 目錄, 並從 github 取下子模組內容.

3.在使用python wsgi.py 啟動動態網站進行編輯，動態內容編輯完成後, 以 generate_pages 轉為靜態內容, 以 git add commit 及 push 將內容推到遠端.
